VSCode 常用快捷鍵

發表於2021-05-16|更新於2021-05-16|其他筆記

一般

  • 命令面板：F1
  • 開啟終端機：CTRL + ~
  • 關閉當前檔案：CTRL + W
  • 搜尋並開啟檔案：CTRL + E
  • 切換已開啟的檔案：CTRL + TAB
  • 存檔：CTRL + S

編輯

  • 複製：CTRL + C
  • 複製該行（不選取）：CTRL + C
  • 剪下該行（不選取）：CTRL + X
  • 貼上：CTRL + V
  • 複製該行至下一行：SHIFT + ALT + 方向鍵下
  • 調整該行順序：ALT + 方向鍵上下
  • 插入下一行：CTRL + ENTER
  • 註解該行：CTRL + /
  • 前往指定行數：CTRL + G
  • 跳至該行首部或尾部：HOME / END
  • 選取：SHIFT + 方向鍵左右
  • 智慧選取：SHIFT + CTRL + 方向鍵左右
  • 多重選取：ALT + CLICK
  • 多重選取相同字符：CTRL + F2
  • 復原上一次多重選取：CTRL + U

插件

  • advanced-new-file：
    • 創建檔案：指令 “Advanced New File” 建議設定自訂快捷建 “CTRL + N”
  • File Utils
    • 重新命名：指令 “File Utils: Rename”
    • 移動檔案：指令 “File Utils: Move”
    • 刪除檔案：指令 “File Utils: Delete”
